## MaCFP-4 Pyrolysis Model Calibration Results (i.e., simulations of idealized material decomposition)
This directory is designed to maintain anaerobic pyrolysis model predictions conducted as part of the MaCFP-4 Pyrolysis Model Calibration Exercise. These computational results are designed for model-to-model comparison, *not* for model validation (i.e., prediction of material decomposition in idealized 0D and 1D heating scenarios). 

Material property sets used to generate these simulations are available in the [Material Properties](https://github.com/MaCFP/matl-db/tree/master/Wood/Material_Properties) folder of this repository. Measurement data (from both mg- and g-scale tests) used to calibrate these property sets have been provided by fire safety science institutions around the world; these results are available on the [Calibration Data](https://github.com/MaCFP/matl-db/tree/master/Wood/Calibration_Data) folder of this repository. If/when using this data and/or related property sets, please cite the appropriate references, as provided in the corresponding README files.
