#### How to Compile the Preliminary Summary of Experimental Measurements 
## (Predecisional Draft Report Submitted to the MaCFP-4 Condensed Phase Workshop)
1. Open and Run `Main_analysis.py` [found in: \..\matl-db\Scripts\MaCFP-4]. Each figure (including those in the Supplemental Information Document) will be saved to the [SCRIPT_FIGURES] folder.
2. Compile the file 'MaCFP-4_Experimental_Summary.tex' [found in: \..\matl-db\Documents\MaCFP-4].

Note: As new data is added to the repo, script figures and tables will update programmatically but the main text (and figure + table captions) will not. The current copy of this report was last compiled on March 25, 2026. As more data is added to this repository, it will be plotted if/when you compile an update copy of the .tex document. A second version of this document will be revised (edits to the main text and additional analysis) in Summer, 2026 to include final results submitted to the MaCFP-4 Workshop.
